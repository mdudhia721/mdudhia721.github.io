PROJECTS

Takubox - Wireless Leverless Controllers | GitHub (WIP)

C++, Embedded Systems, PCB Design, CAD, Product Design/Development

Takubox is a wireless, leverless controller I designed used primarily for competitive fighting games. Some of the features include compatibility with PS5/PC, SOCD Cleaning, Input mode-switching, Input-display OLED, Modular directionals, Hot-swappable keys, and audio support!

Swamp Cooler System | GitHub

Embedded Systems, Low Level C Programming

Arduino-based swamp cooler system. Focused on register-level C++ programming, using GPIO, ADC, and UART to optimize condition monitoring and automated vent/fan control. LCD interface for real-time environmental data and operational state display. Real-time event tracking to log system state transitions and user inputs. You can check out the GitHub for some more details, but this video provides a quick overview as well.
